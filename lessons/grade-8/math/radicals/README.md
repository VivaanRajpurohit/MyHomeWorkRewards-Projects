# Grade 8 Math: Radicals and Complex Numbers

This collection develops radical expressions from basic roots through rational exponents, operations, equations, functions, and complex numbers. Lesson files preserve the original MyHomeworkRewards layout and include revealable worked answers.

Use the numbered lesson filenames in order. The three `practice-*.php` files provide additional word problems for Lessons 1-2, Lesson 5, and Lesson 6.
