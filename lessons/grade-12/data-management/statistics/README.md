# Grade 12 Data Management: Statistics

This collection contains 21 complete lessons across three units:

- `unit-1`: data collection and experimental design
- `unit-2`: counting and probability
- `unit-3`: sampling distributions
- `practice`: cumulative unit reviews
- `data`: structured instructional and question content
- `includes`: shared renderers, navigation, and interactive components

Each public lesson file is a small route wrapper. Content and question data live separately so future lessons can reuse the same textbook-style presentation without duplicating page code.
